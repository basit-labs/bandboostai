Build a production-ready AI-powered English exam preparation platform called "BandBoost AI".

The platform must help users prepare for IELTS and PTE exams by practicing Listening, Reading, Writing, and Speaking. The system must include AI-based feedback, personalized study plans, progress analytics, score prediction, mock tests, podcasts, and adaptive learning recommendations.

The app must first be built as a mobile-first Progressive Web App (PWA) optimized for Android phones. The architecture must also support a future React Native mobile app capable of generating Android APK files using Expo EAS.

The system must be scalable, modular, and production-ready.

------------------------------------------------

PRODUCT GOAL

Create an intelligent exam preparation ecosystem where users can:

Learn English through structured lessons
Practice daily exam tasks
Receive AI feedback
Track progress
Estimate exam readiness
Predict likely exam scores
Take realistic mock exams
Improve weak skills with AI tutoring
Follow a personalized study roadmap

The system should be comparable in quality to modern language learning platforms.
